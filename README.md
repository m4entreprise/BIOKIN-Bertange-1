# Welcome to the BIOKIN project

## Project info

This repository contains the source code for the BIOKIN website.

## Getting started locally

Ensure that Node.js & npm are installed ([install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)).

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm install

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```

You can also edit files directly in GitHub or work inside a GitHub Codespace if you prefer a cloud IDE.

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## How can I deploy this project?

Build the production bundle with `npm run build`. The contents of the `dist/` directory can be deployed to any static hosting provider or integrated into your preferred deployment workflow.
